# Scope snapshots

Timestamped cached reflections of the Atlassian API-token scope picker.

These files are not recommendations and are not authoritative API docs. They
capture what the picker appeared to expose on the date in each filename, for
context while maintaining [../token-scopes.md](../token-scopes.md). Atlassian
can change these lists without notice; verify against the live picker and
official docs before creating a token policy from a snapshot.
